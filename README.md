Consists of 25 minutes work and 5 minute break with a long 10 minute break after 4 work sessions.
  <h2>PreRequisites</h2>
  <p>You will need to have PIL and tkinter installed.</p>
  <p>tkinter is pre-installed in mac and windows but sometimes it doesn't come pre-packaged with Linux so make sure to check if it is installed for you.</p>
  <h3>Installing tkinter for Ubuntu</h3>
  <p>sudo apt-get install python3-tk</p>
  <h3>Installing PIL</h3>
  <p>python3 -m pip install Pillow</p>
